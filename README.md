**SalaoApp**

Aplicativo mobile para gerenciamento de salão/barbearia desenvolvido com React Native e Firebase.

**Funcionalidades**
Login e cadastro
Recuperação de senha
Agendamento de horários
Histórico de atendimentos
Integração com Firebase

**Tecnologias utilizadas**

React Native
Expo
Firebase
JavaScript
